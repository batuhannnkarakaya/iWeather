# iWeather

Şehrinizin ister günlük isterseniz gelecek 5 günün hava derecelerini kontrol edebilrisiniz hava kalitesiyle birlikte.

## Technologies Used

- React
- React-Select
- CSS
- Fetch API
- OpenWeatherMap API
